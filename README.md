# 🥔 Potato Clicker

**The most advanced potato simulation on the internet.**

![potato](https://img.shields.io/badge/potato-soft-yellow)

---

## 🧾 Description

This project does absolutely nothing useful.  
It counts imaginary potatoes. That's it.

No APIs. No databases. No logic.  
Just potatoes.

---

## ✨ Features

- Click a button → potato counter goes up.
- Click another button → potato counter goes down.
- Click a third button → nothing happens (feature).
- Responsive design (potatoes look good on mobile).
- Dark mode (potatoes are easier to see at night).
